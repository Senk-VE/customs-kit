СД | Расчет срока ИМ

Калькулятор срока подачи импортной статистической декларации

Инструмент предназначен для автоматического расчета крайнего срока представления импортной статистической декларации в Республике Беларусь.

Калькулятор учитывает установленный законодательством срок подачи декларации — 7 рабочих дней с даты возникновения обязанности по декларированию. При расчете автоматически исключаются выходные и официальные праздничные дни Республики Беларусь, а также учитываются переносы рабочих дней, установленные законодательством.

В результате пользователь получает точную дату последнего дня, когда статистическая декларация может быть представлена без нарушения установленного срока.

Нормативная база:
• Постановление Совета Министров Республики Беларусь от 14.09.2020 № 533 «О статистическом декларировании товаров».

Актуальность данных:
Расчет выполняется с учетом календаря рабочих дней Республики Беларусь на 2026 год, включая официальные праздничные дни и установленные переносы рабочих дней.

СД | Конвертор валют

Конвертер валют для статистического декларирования

1) Упрощает пересчет стоимости поставок/отгрузок (в том числе при расчетах в различных валютах) в EUR для определения необходимости подачи статистической декларации. Для расчета используется стоимость товаров, а в поле «Дата» указывается дата заключения договора в соответствии с требованиями законодательства.

2) Предназначен для контроля валютных значений при заполнении граф 8, 21, 21а статистической декларации. Позволяет выполнить пересчет валюты на дату отгрузки/поставки, в том числе при указании характера сделки «55» для определения суммы в BYN и формирования статистической стоимости в USD.

Поддерживаемые валюты ввода: BYN, USD, EUR, RUR.
Валюты результата: EUR, USD, BYN.

Нормативная база:
• Постановление Совета Министров Республики Беларусь от 14.09.2020 № 533 «О статистическом декларировании товаров».
• Постановление Государственного таможенного комитета Республики Беларусь от 26.01.2012 № 2 «О статистическом декларировании товаров».

ДТ | Извлечь файлы из XML

Извлечение электронных документов из XML декларации на товары

Инструмент предназначен для извлечения электронных документов, размещенных в информационной системе таможенных органов (ИСТО), из XML-файлов.

Приложение автоматически обрабатывает XML-файл декларации на товары, извлекает вложенные документы и формирует архив ZIP с файлами для дальнейшего просмотра и использования.

Инструмент позволяет получить информацию о размещенных документах, включая их наименование, размер, реквизиты и дату, а также выполнить загрузку отдельных архивов или сформировать единый архив при обработке нескольких XML-файлов.

Основные возможности:
• извлечение электронных документов из XML-файлов деклараций на товары;
• просмотр информации о вложенных документах;
• формирование ZIP-архивов;
• пакетная обработка нескольких XML-файлов.

Конфиденциальность:
Обработка XML-файлов выполняется локально в браузере пользователя. Загружаемые файлы не передаются на сервер и не сохраняются разработчиком.

СД | Контроль данных XML

Контроль и анализ XML-файлов статистической декларации

Инструмент предназначен для автоматического анализа XML-файлов статистических деклараций Республики Беларусь и проверки ключевых показателей перед представлением декларации.

Приложение обрабатывает XML-файл непосредственно в браузере пользователя, извлекает основные сведения о товарах и отображает их в удобном структурированном виде, что позволяет быстро выполнить визуальный контроль данных.

Инструмент отображает сведения о товарах, весовых показателях, количестве, единицах измерения, стоимости по счету-фактуре, статистической стоимости, валюте расчета, а также итоговые значения по декларации.

Основные возможности:
• анализ XML-файлов статистических деклараций;
• отображение сведений по каждому товару;
• контроль веса, количества и единиц измерения;
• отображение фактурной и статистической стоимости;
• просмотр итоговых данных декларации;
• пакетная обработка нескольких XML-файлов.

Конфиденциальность:
Обработка XML-файлов выполняется локально в браузере пользователя. Загружаемые файлы не передаются на сервер и не сохраняются разработчиком.
